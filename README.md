## sudoku-vue （TS版本暂未完成，开发中）
vue版本的数独小游戏。
技术栈采用vue2.5.x + vue-router + vuex

灵感借鉴于慕课网视频。
[从JS到TS开发数独游戏（JS版）](https://www.imooc.com/learn/899)

-----------------------------------

## 安装依赖
```
npm install
```

### 本地运行
```
npm run serve
```

### 线上发布
```
npm run build
```

### 本地测试
```
npm run test
```

### 代码检验
```
npm run lint
```

### 项目依赖vue-cli3.x构建
